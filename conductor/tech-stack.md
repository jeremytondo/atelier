# Technology Stack

## Core Tools
- **Shell:** Zsh (with custom prompt and autosuggestions)
- **Scripting:** Bash (for installation and automation)
- **Configuration Management:** GNU Stow (for modular dotfile symlinking)
- **Persistent Sessions:** `shpool` (for remote VM stability)

## Package Management
- **macOS:** Homebrew
- **Arch Linux:** Yay / Pacman
- **Ubuntu:** `apt`
- **Runtimes:** `nvm` (Node.js)

## Development Environment
- **Editor:** Neovim (configured via LazyVim)
- **Containerization:** Docker & Devcontainers
- **Terminal:** Ghostty (as configured in dotfiles)
